# Админка

Личный кабинет для управления контентом: **`/admin`** (например,
`https://pattaya-kids-guide-web.vercel.app/admin`).

Что умеет:

- **Места** — создание, редактирование (описания ru/en, адрес, координаты,
  тристейт-факты «есть/нет/уточняется», часы работы, категории, видимость),
  обложка, галерея (добавить/удалить фото), удаление.
- **События** — создание/редактирование/удаление, афиша-обложка, привязка к
  месту из каталога или площадка текстом. Даты вводятся по времени Паттайи.
- **Занятия** — создание/редактирование/удаление, цены, возраст (в месяцах),
  обложка. Возрастные классы (таблица Little Gym) пока правятся не здесь.

Все правки видны на сайте сразу (кэш сбрасывается автоматически).

## Вход

Один администратор, вход по паролю. Пароль живёт в переменной окружения
`ADMIN_PASSWORD`:

- **локально** — строка `ADMIN_PASSWORD=...` в `.env` (уже сгенерирован);
- **на Vercel** — Settings → Environment Variables → добавить
  `ADMIN_PASSWORD` (любой свой, от 8 символов) → Redeploy.

После входа браузер запоминает вас на 30 дней (HttpOnly-кука). Смена пароля
в env мгновенно разлогинивает все устройства. Кнопка «Выйти» — в шапке.

Пароль нигде не публикуется: ни в git, ни в чатах. `/admin` закрыт от
поисковиков (robots + noindex).

## Фото

Загружаемые фото автоматически ужимаются до 1600px (jpeg): айфонные
оригиналы по 3–7 МБ сайту не нужны.

Куда сохраняются:

- **локально** — в `public/images/uploads/` (не забудьте потом
  закоммитить+запушить, чтобы файлы попали на прод);
- **на Vercel** — нужно one-time включить Blob-хранилище: в дашборде
  проекта → Storage → Create Database → **Blob** → Connect. Токен
  `BLOB_READ_WRITE_TOKEN` подключится сам, после Redeploy загрузки на
  проде заработают (файлы будут жить в Blob, без коммитов).

## Если что-то пошло не так

- «Неверный пароль» — проверьте `ADMIN_PASSWORD` в `.env` (без кавычек и
  пробелов) и перезапустите `npm run dev`.
- Фото не грузится на проде — скорее всего, не подключён Blob (см. выше).
- Страница `/admin` отдаёт ошибку про `ADMIN_PASSWORD` — переменная не
  задана в окружении.
