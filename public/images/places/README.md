# Фотографии мест (обложки)

Сюда кладём обложку места. Как добавить фото:

1. Положите картинку в эту папку, например `laridea.jpg`
   (подойдёт `.jpg` / `.png` / `.webp` / `.svg`; лучше горизонтальную, ~16:10).
2. В `prisma/seed.ts` у нужного места пропишите путь:
   `imageUrl: "/images/places/laridea.jpg"`.
3. Пересейте базу: `npx tsx prisma/seed.ts`.

Либо просто пришлите фото — я положу файл и пропишу путь.

`sample.svg` — образец для демо-места, можно удалить, когда будут реальные фото.
